# robotic-arm
i am creating a 3d printed robotic arm for stasis! here i will document everything

so basically what the idea is that the robotic arm will be a 4DOF robotic arm which will be like a desk assistant it can be programmed to do many things like
1. picking and placing objects
2. holding objects while i solder them
3. look cool
4. have customized resting positions

## Hardware
| Component | Purpose | Qty |
|---|---|---|
| ESP32 DevKit | Main microcontroller | 1 |
| PCA9685 | 16ch PWM servo driver | 1 |
| DS3218 Servo | Joint actuation | 4 |
| MG90s Servo | Gripper actuation | 1 |
| 5V 10A PSU | Servo power supply | 1 |
| M3 Screws + Brass Inserts | Frame assembly | 1 set 

## Softwares Used
1. Fusion360 for CAD assembly design
2. EasyEda for PCB design (NO AUTOROUTING USED)
